# Parking

---

##  À propos
**Parking** est un projet permettant la **détection et gestion de véhicules via immatriculation**, avec une combinaison de scripts Python et C++.  
Le projet peut servir de base pour un système de contrôle d’accès ou de gestion automatisée d’un parking.

---

##  Structure du projet
final.py → script principal / version finale en Python
*.cpp → fichiers source C++ pour traitement spécifique
README.md → documentation du projet
LICENSE → licence MIT
 

---

##  Fonctionnalités principales
- Détection / gestion d’immatriculations
- Support Python et C++ selon les besoins
- Base pour systèmes de reconnaissance de véhicules
- Scripts modulaires pour analyses ou tests

---

## Installation & Préparation

### Prérequis
- **Python ≥ 3.8**  
- **Compilateur C++ ≥ C++14**  
- Librairies Python (à installer via pip) :

pip install opencv-python numpy
(ajuster selon les dépendances exactes du projet)

--- 

### Lancer le projet
Version Python
bash
Copier le code
python3 final.py
Version C++
bash
Copier le code
# Compiler un fichier C++
g++ main.cpp -o parking_app
./parking_app
